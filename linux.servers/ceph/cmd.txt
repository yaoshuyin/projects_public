.ceph health

.ceph time-sync-status
 查看mon节点时间同步情况
.ceph osd df
 查看osd的使用信息
.ceph osd tree
 查看osd的状态和编号以及分布情况
.ceph osd metadata 0
 查看osd元数据的详细信息
.ceph mon dump
 查看mon的信息
.ceph mon stat

 ceph quorum_status -f json-pretty

 ceph osd pool ls detail
.查看池的IO情况
 ceph osd pool status

 ceph pg dump

 ceph pg map 1.6c

 ceph pg {pg-id} query
